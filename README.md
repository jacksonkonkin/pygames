# Retro-Modern Arcade Collection

A high-performance arcade homepage featuring classic games built with native JavaScript and HTML5 Canvas.

## Web Version (Primary)

The current version of this project is a **100% static web application**. 

- **Homepage**: [web/index.html](web/index.html)
- **Tech Stack**: JavaScript, HTML5 Canvas, Synthwave CSS
- **Deployment**: Optimized for Vercel static hosting

## How to Play

1. Open `web/index.html` in any modern browser.
2. Select a game from the arcade menu.
3. Use the keys specified in the game metadata.

## Old Python Versions

The original Pygame versions have been deprecated in favor of the high-performance Web version for easier accessibility and better deployment.
